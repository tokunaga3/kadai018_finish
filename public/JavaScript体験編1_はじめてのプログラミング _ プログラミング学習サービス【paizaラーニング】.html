      ＃02:はじめてのプログラミング
      受講済みにする

      JavaScript を使ったプログラムを体験します。はじめてなので、簡単なメッセージを表示するプログラムを書いてみましょう。

      メッセージを表示する 

      console.log("Hello world");

      コードの基本的な書き方

      - プログラムでは、命令を使って、コンピュータに動作を指示します
      - console.log() は、データを表示しろという命令です
      - このような命令を、JavaScript では「関数」と呼びます
      - カッコのなかに、表示するデータを指定します
      - カッコのなかのデータを、「引数」(ひきすう)と呼びます
      - 表示する文字をダブルクォーテーションで囲みます
      - コードの行末に「;」を付けます

      メッセージを変える 

      console.log("Hello paiza");

      メッセージを 2 行表示する

      複数行のコードがあると、上から順番に実行する

      console.log("Hello paiza");
      console.log("Hello paiza");

      日本語に変える 

      console.log("Hello paiza");
      console.log("冒険に出かけよう");
